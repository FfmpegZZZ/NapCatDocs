由此可以看出 使用NapCat必须保证NTQQ的客户端这一基础的存在, 于是使用流程大概为

0. 安装**对应版本**NTQQ (特殊情况不需要安装或安装器代为安装)

1. 下载你需要的NapCat版本

2. 遵照其详细教程启动NapCat本体

3. 进入WebUi 启动你需要HTTP/WS 客户端与服务端

4. 配置框架或者独立插件连接NapCat
